Handy [Moment.js](http://www.momentjs.com) filters for your [Vue.js](http://vuejs.org/) project.

## Installation

Install via NPM...

```sh
$ npm install vue-ig-api
```

...and require the plugin like so:

```js
Vue.use(require('vue-ig-api'));
```
